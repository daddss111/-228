Используемые библиотеки:
pygame — библиотека для разработки игр с графикой, управлением и звуком.
random — встроенная библиотека для генерации случайных чисел.
sys — встроенная библиотека для управления системой, используется для выхода из программы.
Основные компоненты и структура кода:
Инициализация:
Инициализация библиотеки pygame.
Создание окна размером 800x600 пикселей.
Настройка цветов для отображения уровней сложности и элементов интерфейса.
Создание шрифтов для отображения текста.
Основные функции и классы:
menu_screen() — отображает меню выбора сложности с тремя кнопками ("Легкий", "Средний", "Тяжелый"). При выборе задается уровень сложности.
game_over_screen() — экран окончания игры с сообщением и приглашением нажать любой клавишу для выхода.
Классы Player, Enemy, Bullet:
Player — управляется стрелками влево/вправо, автоматическая стрельба. Представлен в виде треугольника.
Enemy — враги, спаунятся сверху, движутся вниз, уничтожаются при попадании.
Bullet — пули для игрока и врагов, движутся вертикально и уничтожаются, когда выходят за границы экрана или попадают в цель.
Логика игры (game_loop()):
Настройка параметров сложности, таких как скорость врагов, частота появления и шанс стрельбы врагов.
Основной цикл игры:
Обработка событий (выход из игры).
Автоматическая стрельба игрока с интервалом.
Спавн новых врагов по таймеру.
Враги стреляют с определенным шансом.
Обновление позиций врагов, пуль, врагов стреляют.
Обработка столкновений:
Пули игрока по врагам — удаляют врагов, увеличивая счет.
Вражеские пули по игроку — конец игры.
Отрисовка всех объектов и текста с очками.
Ограничение FPS в 60 кадров.
Основной запуск (main()):
Постоянный цикл: выбор сложности, запуск игрового процесса, отображение экрана окончания игры. После окончания – возвращение к меню выбора.
Итог:
Простая космическая стрелялка с тремя уровнями сложности.
В игре есть управление, автоматическая стрельба, враги, враги стреляют, начисление очков.
Пользователь выбирает уровень сложности через графический интерфейс.
